# React Todo List App

A simple Todo List application built using React.js.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Save tasks using Local Storage
- Tasks remain available after page refresh

## Technologies Used

- React.js
- JavaScript
- CSS
- HTML5

## Project Structure


src/
│
├── App.js
├── App.css
└── index.js


## Installation

1. Clone the repository

bash
git clone https://github.com/your-username/todo-list-app.git


2. Navigate to the project folder

bash
cd todo-list-app


3. Install dependencies

bash
npm install


4. Start the application

bash
npm start


## Usage

- Enter a task in the input field.
- Click *Add* to add a new task.
- Click *Edit* to update a task.
- Click *Delete* to remove a task.
- Use the checkbox to mark a task as completed.
